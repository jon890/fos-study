# 카드 작성 미니 템플릿

SKILL.md 의 Step 4 (작성) 단계에서 참조한다. 카드 유형별 표준 형식.

## 신규 구현 카드

```
[제목]   X 시스템 — 신규 구축
[기간]   YYYY.MM - YYYY.MM | 팀명

[주요 업무]
- 무엇을 만들었는가 (1줄)
- 어떤 구조로 설계했는가 (1줄)
- 어떤 의사결정을 내렸는가 (1-2줄)

[주요 성과]
- 측정 단위 명시한 정량 (1-2줄)
- 운영·확장 가치 (1줄)

[사용한 기술] 핵심만 나열
```

## 트러블슈팅 카드 (4블록)

```
[제목]   X 장애 / Y 안정화 / Z 최적화
[기간]   YYYY.MM | 컨텍스트

[문제]   어떤 현상 + 어떻게 인지
[원인]   어떻게 분석 + 어떤 구조로 식별
[해결+검증]   어떤 계획 + 어떻게 검증
[결과]   무엇이 달라졌고 재발 방지

[사용한 기술] 핵심만
```

## 진화·리팩터링 카드

```
[제목]   X 점진 리팩터링 / 도메인 추상화

[주요 업무]
- 시작 상태 (어떤 중복·문제가 있었는가)
- 어떤 단계로 진화시켰는가 (단순화한 톤)
- 팀 전파·정착

[주요 성과]
- 정량 (분량 / 기간 / 후속 효과)
- 추상화 후 후속 작업이 얼마나 쉬워졌는가
```

## 보조 카드 (짧게)

```
[제목]   X 구현
[기간]   YYYY.MM

[주요 업무]
- 한두 줄로 핵심
- 사용한 핵심 기술 1줄

[사용한 기술] 짧게
```
