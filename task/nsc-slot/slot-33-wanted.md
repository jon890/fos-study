# Slot 33 (Wanted) 개발

**진행 기간**: 2024.10 ~ 2024.12

---

## 배경

Wanted는 링크게임 메커니즘이 핵심인 슬롯이다. 일반 스핀에서 링크 심볼을 모아서 윈도우를 채우면 링크게임에 진입하고, 링크게임에서 큰 보상을 얻는 구조다.

Slot 21보다 구조가 복잡해서 구현 기간이 길었다.

---

## 핵심 메커니즘

### 링크게임 (Link Game)

일반 베이스 스핀에서 링크 심볼이 등장할 때마다 보드에 쌓인다. 특정 조건(디스크 배수 X 이상, 링크 심볼 일정 수 이상)을 만족하면 링크게임에 진입한다.

링크게임에서는 릴이 새로 돌아가면서 추가 링크 심볼을 모을 수 있다. 윈도우가 링크 심볼로 가득 차면 Grand Jackpot이 발생한다.

```
베이스 스핀
  → 링크 심볼 등장 → 보드에 고정
  → 조건 충족 시 링크게임 진입

링크게임
  → 스핀 진행 (링크 심볼 위치는 고정)
  → 추가 링크 심볼 등장 → 보드에 추가
  → 새 심볼 없는 스핀 3연속 → 링크게임 종료
  → 윈도우 가득 차면 → Grand Jackpot
```

### 디스크 배수 (Disk Multiplier)

링크 심볼에는 배수값이 붙어 있다. 링크게임 종료 시 모인 링크 심볼의 배수값 합계가 최종 보상이 된다.

디스크 배수가 일정 기준 이상이어야 링크게임에 진입할 수 있도록 진입 조건이 있었다.

### 텀블링

Slot 21과 마찬가지로 텀블링 메커니즘을 사용한다. 당첨 심볼이 제거되고 위에서 새 심볼이 채워진다.

---

## 구현하면서 있었던 일들

**베이스에서 처리된 심볼이 다음 텀블에서 재처리되는 버그**

텀블링 로직에서 이미 처리된 심볼이 다음 텀블 사이클에서 다시 처리 대상으로 잡혔다. 처리 완료 심볼을 별도로 추적해서 제외시켰다.

**base → link 진입 시 windowHeight 불일치**

링크게임 진입 시 창 높이(windowHeight) 값이 맞지 않는 현상이 있었다. 베이스 스핀과 링크게임의 윈도우 크기 계산이 달랐다.

**링크게임 early 종료 조건**

링크 심볼이 윈도우를 가득 채우면 즉시 종료되어야 하는데, 종료 조건 검사 순서가 잘못되어 있었다. 조건 검사를 스핀 결과 처리 전으로 앞당겼다.

**하이 심볼 배지 (High Symbol Badge)**

특정 고가 심볼에 배지가 붙는 기능인데, 배지 처리 조건이 스펙과 달랐다. 조건 검사 로직을 스펙에 맞게 수정했다.

**보너스 심볼 예외 처리**

보너스 심볼이 예상치 못한 위치에 등장할 때 예외가 발생했다. 예외를 던지지 않고 로그만 남기도록 처리했다.

**링크 심볼 초기 윈도우 대응**

게임 최초 진입 시 이미 윈도우에 링크 심볼이 있는 케이스를 처리하지 못했다. 초기 윈도우 설정 시 링크 심볼 포함 케이스를 추가했다.

**베이스 → 링크 진입 시 BLANK 처리**

링크게임 진입 시 다음 상태의 릴에서 링크 심볼이 없는 자리는 BLANK로 채워야 하는데, 링크 심볼 위치를 제외하지 않고 전체를 BLANK로 채우고 있었다.

**바이피처(BuyFeature) 대응**

바이피처로 링크게임에 바로 진입하는 케이스 대응. 최소 디스크 배수 조건을 바이피처 시에는 다르게 적용해야 했다.
