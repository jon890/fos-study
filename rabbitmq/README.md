# RabbitMQ

RabbitMQ 학습 허브. 이 폴더는 RabbitMQ 자체 기본기와 Kafka 대비 선택 기준 문서를 묶는다.

현재 문서 역할:
- [RabbitMQ Basics](./rabbitmq-basics.md) — Exchange, Queue, Binding, Ack, Prefetch, DLQ 등 기본기와 실전 설정
- [RabbitMQ vs Kafka](./rabbitmq-vs-kafka.md) — 메시징 기술 선택 기준과 운영 관점 비교

권장 읽기 순서:
1. `rabbitmq-basics.md`로 기본 구조와 운영 포인트 파악
2. `rabbitmq-vs-kafka.md`로 어떤 상황에서 RabbitMQ를 고르는지 정리

앞으로 문서가 늘어나면 이 README를 허브로 유지하고, 개별 문서는 기본기 / 비교 / 운영 / 실수 패턴 / 사례 문서로 역할을 나눠 확장한다.
