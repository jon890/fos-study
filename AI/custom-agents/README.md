# 자주 사용하는 나만의 에이전트를 정리해보자.

- 클로드 커맨드, 스킬을 써보고 싶은데, 클로드 코드에 종속되는 문제가 있다.
- 팀원들이 여러 AI 도구를 사용하고 있고, 체계화된 에이전트 워크플로가 필요할 때 사용하도록 정리해본다.

## 커밋-푸시 에이전트

- [링크](./commit-and-push-agent.md)

### 역할

- 테스트 수행
- 의미 단위의 파일 스테이지 (한 커밋에 여러 의미를 담지 않도록)
- 커밋 메시지 작성
- 커밋 메시지를 사용자에게 승인받은 후, 커밋 진행
- 푸시를 할지 사용자에게 승인받은 후, 푸시 진행

### 하지 말아야 할 것

- 절대 `force push` 하지 않는다
